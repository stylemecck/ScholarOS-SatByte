# 🎓 Student Toolkit Pro - AI-Powered Academic Success Hub

![Student Toolkit Pro Banner](https://images.unsplash.com/photo-1523240795612-9a054b0db644?auto=format&fit=crop&q=80&w=1200&h=400)

**Student Toolkit Pro** is a cutting-edge, MERN-stack platform designed to empower competitive exam aspirants and college students. Built with a "Mobile-First" philosophy and powered by **Gemini 3.0 Flash AI**, it provides highly accurate rank predictions, intelligent study planning, and professional resume building—all in one place.

---

## 🚀 Key Features

### 🤖 AI-Powered Analysis
- **AI Rank Predictor**: Data-driven rank and college predictions based on historical shift-wise normalization.
- **Marks vs Percentile**: Intelligent conversion of raw marks into estimated percentiles factoring in exam difficulty.
- **AI Study Planner**: Custom 7-day study roadmaps generated by AI to keep you on track.
- **AI Resume Magic**: Professional summary generation and bullet-point enhancement for high-impact resumes.

### 🛠️ Academic Utilities
- **CGPA Calculator & Converter**: Effortless grade management based on university standards.
- **Attendance Tracker**: Stay on top of your 75% requirement with smart "How many more classes?" logic.
- **PDF to Text**: Extract content from academic papers with high accuracy.

### 💰 Secure Ecosystem
- **Personal Dashboard**: Save and track your AI analysis history securely in MongoDB.
- **Credit System**: Paid AI credits integrated with **Razorpay** for a seamless premium experience.
- **Modern UI**: Sleek, high-fidelity design built with Tailwind CSS v4 and Framer Motion.

---

## 🛠️ Tech Stack

- **Frontend**: React (Vite), Tailwind CSS v4, Framer Motion, Lucide React, Axios
- **Backend**: Node.js, Express.js, MongoDB (Mongoose)
- **AI Engine**: Google Gemini 3.0 Flash (Generative AI SDK)
- **Payment Gateway**: Razorpay
- **PDF Processing**: PDF-Parse, Python (Playwright for PDF Generation)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/student-toolkit-pro.git
   cd student-toolkit-pro
   ```

2. **Backend Setup**:
   ```bash
   cd backend
   npm install
   # Create a .env file and add:
   # PORT=5000
   # MONGO_URI=your_mongodb_uri
   # JWT_SECRET=your_secret
   # GEMINI_API_KEY=your_key
   # RAZORPAY_KEY_ID=your_id
   # RAZORPAY_KEY_SECRET=your_secret
   npm run dev
   ```

3. **Frontend Setup**:
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
